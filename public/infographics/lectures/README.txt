Drop infographic images for each lecture in this folder.
Naming convention: <LectureID>.<ext>

Supported formats: .png  .jpg  .jpeg  .webp  .svg

Examples:
  L01.png   → appears on the L01 (Introduction) lecture page
  L02.jpg   → appears on the L02 lecture page
  L14.webp  → appears on the L14 lecture page

The page automatically picks up the file — no code changes needed.
