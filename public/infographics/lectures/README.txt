Drop infographic images into one folder for each lecture.
Folder convention: <LectureID>/

Supported formats: .png  .jpg  .jpeg  .webp  .svg

Examples:
  L01/overview.webp        -> appears on the L01 lecture page
  L03/1-harris.webp        -> appears first on the L03 lecture page
  L03/2-corner-score.webp  -> appears second on the L03 lecture page

The page automatically finds every supported image in the matching lecture
folder, sorts the filenames naturally, and displays them as a gallery.
