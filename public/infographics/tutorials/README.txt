Drop infographic images for each tutorial in this folder.
Naming convention: <TutorialID>.<ext>

Supported formats: .png  .jpg  .jpeg  .webp  .svg

Examples:
  od-01.png    → appears on Object Detection tutorial 1 page  (od-01 … od-09)
  ssl-03.webp  → appears on Self-Supervised Learning tutorial 3 page  (ssl-01 … ssl-10)
  tr-07.jpg    → appears on Tracking tutorial 7 page  (tr-01 … tr-08)

The page automatically picks up the file — no code changes needed.
